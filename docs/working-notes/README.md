# Working Notes and Internal Documentation

This directory contains internal working documents, development notes, and status tracking files that are not part of the main project documentation.

## Contents

- **ARCHITECTURE_*.md** - Architecture analysis and design documents
- **CLIENT_*.md** - Client development and refactoring notes
- **CONVERTER_*.md** - Converter development and reference materials
- **ENVIRONMENT_*.md** - Environment setup and troubleshooting guides
- **RELEASE_*.md** - Release checklists and status tracking
- **PRODUCTION_*.md** - Production test status and tracking
- **INTERNAL_*.md** - Internal usage reports and documentation
- **DOMAIN_STATUS/** - Domain-specific development status files

These files are excluded from the package distribution but kept for development reference.
