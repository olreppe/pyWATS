# API Specifications

This directory contains detailed API specifications for the pyWATS API.

## Structure

- `endpoints/` - Individual endpoint specifications
- `authentication.md` - Authentication and authorization specs
- `error_handling.md` - Error handling specifications
- `versioning.md` - API versioning strategy

## Format

Specifications should follow a consistent format:

```markdown
# Endpoint Name

## Overview
Brief description of the endpoint's purpose.

## HTTP Method and Path
`GET /api/v1/example`

## Parameters
### Path Parameters
### Query Parameters
### Request Body

## Responses
### Success Response
### Error Responses

## Examples
### Request Example
### Response Example
```
