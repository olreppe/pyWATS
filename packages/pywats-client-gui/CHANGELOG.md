# Changelog

All notable changes to pywats-client-gui will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - TBD

### Added
- Modern PySide6-based Qt interface
- Tab customization (show/hide tabs)
- Integrated debug logging in GUI
- Multiple instance support with separate configurations
- Enhanced converter management UI
- Real-time queue visualization
- Log viewer with search and filtering
- Connection status indicators
- Software management tab (optional)
- SN Handler tab (optional)
- Keyboard shortcuts for common actions
- Export functionality (logs, queue, reports)
- System tray integration
- Auto-start support

### Changed
- Migrated from PyQt5 to PySide6 (Qt6)
- Redesigned UI for better usability
- Improved converter testing interface
- Enhanced log viewing capabilities
- Better status reporting

### Fixed
- UI freezing during long operations
- Tab layout issues on different screen sizes
- Dark mode compatibility
- High DPI display issues

## [1.0.0] - Initial Release

### Added
- Initial GUI release
- Basic Qt interface
- Connection management
- Converter list
- Queue monitoring
- Log viewing
