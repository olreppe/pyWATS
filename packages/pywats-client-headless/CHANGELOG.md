# Changelog

All notable changes to pywats-client-headless will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - TBD

### Added
- Full CLI interface with subcommands
- HTTP control API for remote management
- Daemon mode for background operation
- Systemd service integration
- Docker deployment support
- Raspberry Pi optimized setup
- Configuration management via CLI
- Connection testing commands
- Converter management commands
- Status monitoring and reporting

### Changed
- Refactored from legacy client to modern CLI/API architecture
- Improved daemon process management
- Enhanced HTTP API with more endpoints
- Better error handling and reporting

### Fixed
- Daemon startup issues on some Linux distributions
- HTTP API binding on all interfaces
- Configuration path handling

## [1.0.0] - Initial Release

### Added
- Initial release of headless client
- Basic CLI interface
- Simple daemon mode
- HTTP API (basic)
