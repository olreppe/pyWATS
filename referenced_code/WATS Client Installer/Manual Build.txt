Manual build should only be used for testing purposes.
!!NEVER distribute manually built installer

Installer intended for distribution must be built using Visual Studio Team Services Build and Release

Manual build instructions (replace *** with actual password for codesigning pfx file)
1) Get latest from source
2) From solution root folder (branch root):
	Execute .\Tools\BuildSupport\ClientRegisterVersion.ps1 -Product "WATS Client" -ProductVersion "5.0" -BuildNumber "5.0.0.0" -BuildType "Local" -Branch "Main" -sign "***" 
3) Build WATS Client solution
4) Execute .\Tools\BuildSupport\PrepareClientInstallerArtifacts.ps1 -Configuration Debug -BuildNumber "5.0.0.0" -sign "***"
5) Build Client installer solution for both x64 and x86 platforms
6) Execute .\Tools\BuildSupport\ClientPostInstaller.ps1 -Destination . -BuildNumber "5.0.0.0" -sign "***"
