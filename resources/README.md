# Resources

This directory contains various project resources including schemas, examples, and configuration templates.

## Structure

- `schemas/` - JSON schemas, OpenAPI specifications
- `examples/` - Example requests, responses, and data
- `configs/` - Configuration file templates

## Schemas

JSON schemas and OpenAPI specifications should be stored in the `schemas/` directory. These provide:

- Request/response validation
- API documentation generation
- Client code generation
- Contract testing

## Examples

The `examples/` directory contains:

- Sample API requests and responses
- Example data for testing
- Use case scenarios
- Integration examples

## Configurations

Template configuration files for:

- Development environment
- Production environment  
- Testing environment
- Docker configurations
