# Active Projects

This folder contains active, in-progress projects for pyWATS development.

## Structure

Each project should be in its own subfolder with:
- `README.md` - Project overview and status
- `PROGRESS.md` - Detailed progress tracking
- Tests, scripts, and documentation as needed

## Lifecycle

1. **Active** - Projects in progress live here
2. **Complete** - Moved to `docs/internal_documentation/completed/{quarter}/`
3. **Archived** - Cancelled/superseded projects go to `docs/internal_documentation/archive/`
