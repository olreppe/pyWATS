# pyWATS API

A Python-based WATS (Web Application Testing System) API.

## Project Structure

```
pyWATS/
??? docs/                   # Documentation and specifications
?   ??? api/               # API specifications
?   ??? architecture/      # Architecture documents
?   ??? requirements/      # Requirements documentation
??? specs/                 # Technical specifications
?   ??? api_specs/        # API endpoint specifications
?   ??? data_models/      # Data model specifications
?   ??? workflows/        # Workflow specifications
??? resources/            # Project resources
?   ??? schemas/          # JSON schemas, OpenAPI specs
?   ??? examples/         # Example requests/responses
?   ??? configs/          # Configuration templates
??? src/                  # Source code (to be created later)
??? tests/                # Test specifications and code
??? requirements.txt      # Python dependencies
??? pyproject.toml       # Project configuration
```

## Getting Started

This project is currently in the specification phase. All specifications and requirements are being documented before implementation begins.

## Specifications

- **API Specifications**: Located in `specs/api_specs/`
- **Data Models**: Located in `specs/data_models/`
- **Workflows**: Located in `specs/workflows/`
- **Schemas**: Located in `resources/schemas/`

## Documentation

Detailed documentation can be found in the `docs/` directory.
